Name:
Natural Earth II
Description:
This is a higher-resolution version of the Natural Earth II tileset that is included in Cesium itself. It was created from the 1:10 million scale Natural Earth II Shaded Relief, Water, Drainages image. This imagery is most useful for applications where the viewer is in space, though some pixelation is noticeable when viewing from low-Earth oribit.

Using this tileset in Cesium

// In an application that uses CesiumWidget:
var widget = new Cesium.CesiumWidget({
    imageryProvider : new Cesium.TileMapServiceImageryProvider({
        url : '//cesiumjs.org/tilesets/imagery/naturalearthii'
    })
});

// In an application that uses Viewer:
var viewer = new Cesium.Viewer({
    imageryProvider : new Cesium.TileMapServiceImageryProvider({
        url : '//cesiumjs.org/tilesets/imagery/naturalearthii'
    }),
    baseLayerPicker : false
});

// In other applications:
var scene = ...;
scene.imageryLayers.removeAll(); // optional
scene.imageryLayers.add(new Cesium.TileMapServiceImageryProvider({
    url : '//cesiumjs.org/tilesets/imagery/naturalearthii'
}));
